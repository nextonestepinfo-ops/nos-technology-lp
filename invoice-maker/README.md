# NosTechnology 書類メーカー（見積・請求・納品）

スマホ／PCのブラウザだけで動く、見積書・請求書・納品書の作成アプリ。
インストール不要。`index.html` を開くだけ。

## 特長
- 📥 **見積書を撮影 → 自動で請求書化**（OCR / AI 読み取り）
- ✏️ **モバイル最適化の明細カード編集**（並べ替え・追加・削除）
- 🔄 **書類タイプ切替**（請求書 / 見積書 / 納品書）＋ 番号自動採番
- 🧾 値引き・源泉徴収（10.21%）・軽減税率8% 対応
- 🏦 振込先・印鑑・会社情報をこの端末に保存（次回自動入力）
- 🕘 履歴（保存・再編集・複製）
- 🖨️ **PDF出力** / 📊 **Excel(.xlsx)出力**
- 🗂️ ファイル名を自動命名：`日付_種別_宛先_担当_番号`
  （例：`2026-06-29_請求書_株式会社○○_担当山田_INV-202606-001`）
  → ドライブ上でAI/手動で整理しやすい

## 読み取りエンジン（設定画面で選択）
| エンジン | 精度 | 費用 | 対応 |
|---|---|---|---|
| OCR.space | 標準 | 無料（デモキー内蔵／自分のキーも設定可） | 画像・PDF |
| OpenAI GPT-4o mini | 高 | 従量（要APIキー） | 画像 |
| Google Gemini | 高 | 無料枠あり（要APIキー） | 画像 |

## APIキーについて（重要）
APIキーは**コードに含めず**、設定画面から入力して**この端末（ブラウザのlocalStorage）にのみ保存**されます。
そのため本リポジトリを **GitHubに公開しても鍵は漏れません**。
内蔵の OCR.space キーは公開デモキー `helloworld`（共有・低速）です。実用時はご自分のキーを設定してください。

## 使い方
1. `index.html` をブラウザで開く
2. 「設定」で会社情報・振込先・印鑑・読み取りエンジンを保存（初回のみ）
3. 「取込」で見積書を読み込む or 「手入力で新規作成」
4. 「作成」で内容を確認・修正
5. 「プレビュー」→ PDF / Excel 出力
