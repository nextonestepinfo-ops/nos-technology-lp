# 素材生成プロンプト（日本語・上から順にコピペ用）

各プロンプトは**スタイル込みの自己完結型**。画像生成AI（ChatGPT/DALL·E・Midjourney・Firefly・Imagen 等）に**上から順に1つずつ貼る**だけでOK。
※ 画像生成は英語の方がやや精度が出やすいツールもあります。うまく出ない時は同名の英語版 `ASSET-PROMPTS.md` を使ってください。

ブランド：ニアブラック #07080C／ネイビー #16213A／エレクトリックブルー #1A5CFF・#3D8BFF／白 #F4F5F8

---

## Webサイトも渡す？（使い分け）
- **画像を作る時**：URLのスクショを1枚“参考画像”として一緒に渡すと雰囲気が揃いやすい（対応ツールのみ）。URL → https://nextonestepinfo-ops.github.io/nos-technology-lp/
- **Codex等にコード作業を頼む時**：リポジトリを渡す → `https://github.com/nextonestepinfo-ops/nos-technology-lp`（「HANDOFF.md を読んで続けて」）。
- ※ 画像“生成”はCodex不可（コード用）。画像は画像生成AIへ。

## STEP 0 — 最初に1回だけ貼る（ブリーフィング）
```
これから「Nos Technology」という、地域店舗（美容室・カフェ・クリニック等）向けにWeb制作・AI自動化・管理画面を作る制作スタジオの、Webサイト用の画像素材を作ります。
全画像に共通する条件：
・雰囲気＝シネマティックで高級感、最新AAAタイトルのキーアートのような照明。自信があり信頼できる印象。
・配色＝ニアブラック#07080Cとネイビー#16213Aを基調に、エレクトリックブルー#1A5CFF/#3D8BFFを差し色。白背景指定の時は#F4F5F8。
・光＝ボリューメトリックで柔らかいリムライト、軽いブルーム、微細なフィルムグレイン、浅い被写界深度。
・文字・ロゴ・透かしは入れない。オーバーレイ用の余白を残す。
・人物＝顔が特定できない構図（手元・後ろ姿・シルエット）。肖像権に配慮。
・超高精細・高品質・アーティファクトなし。
以降、1つずつ画像の指示を送るので、上記の共通条件を必ず守って生成してください。
```

## STEP 1 — OGP / SNSシェア画像（1200×630, WebP）→ assets/og/og-share.webp
```
テック系スタジオのブランドキービジュアル。右側に、発光する低ポリゴンの結晶状AIコア（球体）が浮かび、ニアブラック#07080Cからネイビー#16213Aへのグラデーション背景。薄い幾何学ネットワーク線と微粒子、エレクトリックブルー#1A5CFFのボリューメトリックな発光。左側は見出しを後で重ねるため大きく余白。シネマティック、AAAゲームのキーアート風、文字・ロゴなし、超高精細。
```
