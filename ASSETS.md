# Nos Technology — 素材リスト（制作仕様書）

サイトのクオリティを“圧倒的”に上げるための素材一覧。
**優先度**: ★★★=必須 / ★★=推奨（質が大きく上がる） / ★=任意（さらに上）
配置先は現状の `index.html` / 各 `concepts/` と対応。仮素材（グラデーション・Canvas描画・SVG再現）を実素材へ差し替える前提。

> 多くはAIで生成できます（ロゴ同様）。各項目に **AIプロンプト例** を添えています。
> 書き出し方針: 写真=WebP(+JPEGフォールバック) / 図版=SVG / 透過=PNG / 動画=MP4(H.264)+WebM。Retina対応で表示サイズの2倍を基準。

## 11. テキスト素材（コピー・情報） ★★★
画像以外の“素材”。これが揃うと一気に本物に。

- 会社情報: 正式社名 / 所在地 / 連絡先 / 設立 / 代表者
- 実績の説明文（各作品 1〜2文）と成果（許諾範囲で数値）
- **お客様の声 / 推薦文**（氏名・店舗名・写真）★効果大
- FAQ（5〜8項目）
- 料金の詳細（3万円プランの内訳・オプション）
- 特定商取引法表記 / プライバシーポリシー（決済・申込みを伴う場合）★必須要件
- メタ情報: ページ説明文（SEO用 120字程度）

## 命名規則（推奨）
- 小文字＋ハイフン: `logo-mark.svg`, `og-default.png`, `work-beauty-salon.webp`
- 倍率: `@2x` を付与（例 `work-diner@2x.webp`）
- 連番は2桁: `work-01`, `service-02`

## 優先着手の3点（最小で効果大）
1. **ロゴ実データ＋ファビコン＋OGP**（★★★ 公開の最低ライン）
2. **Worksの実サムネ5枚**（説得力。デモ作品でも可）
3. **管理画面UIスクショ4枚**（ヒーロー3Dパネルが本物化）
